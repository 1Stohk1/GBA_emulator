# GBA_emulator
Emulatore del GameBoy Advance in C

## Building
To build the emulator (Headless by default):
```bash
make
```
To build with graphics (requires SDL2):
```bash
make SDL=1
```

## Running
Run with a ROM file:
```bash
./gba_emu zaffiro.gba
```
